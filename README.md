# Propositional Logic Truth Table Solver

A web-based interactive tool built with HTML, CSS, and JavaScript that evaluates compound propositional logic expressions and generates a clean, dynamic truth table.

## 📌 Problem Description
This project solves the following propositional logical expressions for given boolean variables $p$, $q$, and $r$:

1. **Expression A:** $p \lor \neg q$
2. **Expression B:** $(p \land q) \rightarrow \neg r$
3. **Expression C:** $(q \rightarrow \neg r) \leftrightarrow p$

---

## 🛠️ Tech Stack
* **HTML5** - Page structure
* **CSS3** - Custom styling and responsive table layout
* **JavaScript (ES6)** - Logic evaluation & dynamic DOM rendering

---

## 🚀 Features
- Automatically generates all $2^3 = 8$ truth combination rows.
- Dynamic color highlight for `1` (True) and `0` (False) values.
- Clean and modern user interface.
